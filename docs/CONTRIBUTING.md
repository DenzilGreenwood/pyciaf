1. __AI Usage & Disclosure Policy__
   
    In alignment with our Research Portfolio, we allow the use of AI assistants (e.g., GitHub Copilot, ChatGPT) under the following conditions:

   - __Human-in-the-Loop:__  All AI-generated code must be reviewed, refactored, and validated by the contributor.

   - __Mandatory Attribution:__ If a significant block of code (e.g., a new Industry Adapter ) is AI-assisted, you must include an __AI Assistance Disclosure__ in the PR description, similar to the one found in the CIAF Research Portfolio.
